Frequency-Domain Electromagnetics#

In this chapter, we publish code comparisons and validations for frequency-domain electromagnetic modeling packages. In SimPEG, the SimPEG.electromagnetics.frequency_domain module is used for modeling frequency-domain data.

Content:

  • Forward simulation of inductive source FEM data for a 1D layered Earth (conductive only)

  • Forward simulation of inductive source FEM data for a 1D layered Earth (conductive and susceptible)

  • Forward simulation of inductive source FEM data for a conductive sphere in a vacuum

  • Forward simulation of inductive source FEM data for a conductive and susceptible sphere in a vacuum
